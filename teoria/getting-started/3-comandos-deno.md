
## VERSION

    - Saber version de deno

        $ deno --version

        $ devo -v

## EJECUTAR DENO

    - Ejecutar archivo local

        $ deno run main.ts

    - Ejecutar archivo remoto

        $ deno run https://mydomain.com/main.ts

    - Alternativa 2 para ejecutar archivo local

        $ cat main.ts | deno run
